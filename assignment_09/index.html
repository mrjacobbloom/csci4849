Assignment 9: AAC Experiment

CSCI 4849-5849 Jacob Bloom

What is this?

This is an experiment in Augmentatitive and Alternative Communication (AAC). It simulates eye tracking via cursor position and uses dwell to determine clicks. The idea is to type a message to be spoken later. Then, while it's being spoken, use a slider in real time to adjust the tone of the following word. The hope is that this finer-grained control over prosody will make it possible to express a more diverse range of emotion via an AAC system.
